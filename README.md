# Delhi Load Forecasting AI — Frontend

Next.js + TypeScript + Tailwind CSS starter scaffold with a ShadCN-style UI component set and clean architecture.

## Quick Start

1. Install dependencies:

```powershell
npm install
```

2. Run the dev server:

```powershell
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Project Layout

- `app/` — Next.js App Router routes and layouts
- `components/ui/` — Reusable UI primitives (Card, Button, Input, etc.)
- `components/dashboard/` — Shell components (TopNav, Sidebar, DashboardShell)
- `lib/api/` — API helpers
- `lib/utils/` — Utility helpers
